#Easy Expense Tracker (In progress)
Over 50% of young people have some kind of loan. The problem lies in the system. 
We can see more and more subscription plans, Pay After services, credit cards over debit cards.

If it's system problem, we need a system solution. To overcome this problem we need simple, user friendly, independent and safe and trusted application.
EET will be coming with features like yearly and monthly expenses graphs, calculations how many money an user can save, creating a saving plan.

To use the application, you need to download your bank history and upload it, the app will handle the rest.
